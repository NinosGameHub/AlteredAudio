`Button` is the text button in three weights; use `primary` (filled blue) for the single main action, `secondary` (sunken grey) for neutral actions, `ghost` for low-emphasis inline actions.

```jsx
<Button variant="primary" pill>Add Module</Button>
<Button variant="secondary">Save Preset</Button>
<Button variant="ghost" size="sm">Reset</Button>
```

- Use `pill` for Apple-style fully-rounded CTAs; default is the 6px card radius.
- Only one `primary` per view — the brand leans restrained.
